Como centralizar uma div usando Flexbox?

João Dias • há 20 minutos

Estou com dificuldades para centralizar uma div tanto horizontalmente quanto verticalmente utilizando CSS. Alguém pode me ajudar?

Respostas (2)

Maria Souza há 10 minutos

Você pode utilizar display: flex, juntamente com justify-content: center e align-items: center. 

.container{
    display: flex;
    justify-content: center;
    align-items: center;
}
Pedro Lima há 5 minutos

Não esqueça de definir uma altura para o elemento pai, como por exemplo height: 100vh, caso queira centralizar verticalmente na tela.
